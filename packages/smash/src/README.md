# SMASH — Schreibblockaden-Brecher

SMASH ist ein Tool zur gezielten Unterbrechung von Schreibblockaden.
Es generiert keinen Text, sondern stellt die Schreibfähigkeit durch
kurze, gezielte Interventionen wieder her.

## Ecosystem

| Tool | Ebene | Funktion |
|------|-------|----------|
| FLOW | Repair/Normalization | Deterministische Textkorrektur |
| SPIN | Satz/Diagnose | Strukturdiagnose & Chunk-Analyse |
| SMASH | Blockade-Lösung | Mikro-Interventionen |

## Status

Konzept-/Prototyping-Phase. Der Prototyp (`smash_warioware_minimalhud_nocountdown.html`)
ist ein WarioWare-inspiriertes Mikro-Spiel-Framework für Schreibblockaden-Interventionen.

## Prinzipien

1. Blockade-Orientierung statt Feature-Sammlung
2. Niedrige Einstiegshürde
3. Sprachnahe Interaktion
4. Kurze Dauer, klarer Effekt
5. User bleibt Autor
6. Kein blindes Raten
